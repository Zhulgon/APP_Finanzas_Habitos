# V1.3 Commercial Sprint Plan

Fecha base: `2026-03-13` (America/Bogota)

## Objetivo

Convertir la app en un producto listo para crecimiento comercial con foco en:

- retencion diaria,
- experiencia premium,
- arquitectura preparada para nube,
- calidad de release y monitoreo.

## Sprint 1 - Core de producto comercial

- [x] Auth base por correo (magic code local en prototipo) + modo invitado.
- [ ] Auth productivo real (email magic link por proveedor externo u OAuth).
- [ ] Perfil cloud con sincronizacion multi-dispositivo.
- [x] Cola de sincronizacion offline-first (enqueue + flush local).
- [ ] Politica de privacidad, terminos y consentimiento en app.

Criterio de salida:

- login funcional en web/mobile,
- datos consistentes al abrir en 2 dispositivos,
- no perdida de datos en reconexion.

## Sprint 2 - Academia financiera diaria (gamificada)

- [x] Lecciones ordenadas por ruta diaria.
- [x] Desbloqueo progresivo: 1 capsula por dia.
- [x] Bloqueo explicito y fecha estimada de siguiente desbloqueo.
- [x] Contenido expandido por pilares financieros.
- [x] Integracion API educativa (pulso USD/COP) con fallback.
- [ ] Quiz corto por leccion (3 preguntas).
- [ ] Repeticion espaciada para reforzar conceptos.

Criterio de salida:

- usuario completa una leccion al dia y vuelve por la siguiente,
- progreso visible y entendible desde la pantalla Aprender.

## Sprint 3 - Analitica de producto y UX premium

- [ ] Embudo onboarding + eventos de retencion D1/D7.
- [ ] Dashboard con objetivos semanales y riesgo de abandono.
- [ ] Notificaciones inteligentes por contexto.
- [ ] Mejoras UI avanzadas (micro-interacciones y vacios guiados).

Criterio de salida:

- se puede medir conversion y retencion semanal en tablero.

## Sprint 4 - Release comercial

- [ ] Monitoreo de errores (crash reporting) y alertas.
- [ ] Suite E2E en flujo critico.
- [ ] Plan freemium/suscripcion.
- [ ] Checklist legal y de tiendas (Play/App Store/Web deploy).

Criterio de salida:

- release candidate listo para pilotos reales.

## Implementado hoy (avance real)

1. Motor de `learningPath` diario.
2. Pantalla `Aprender` redisenada para flujo diario.
3. Catalogo educativo ampliado a 10 lecciones.
4. Integracion `financialPulse` (USD/COP) con degradacion segura.
5. Tests unitarios nuevos para motor de aprendizaje y pulso financiero.
6. Base de autenticacion y sesion persistente.
7. Cola de sincronizacion local y panel de control en Perfil.
